Answers to Questions

1. How many factors should she select? Why? Why not more or less?

We have selected 5 factors for three main reasons:

  1. There are 5 factors with eigenvalue > 1.00 (the sixth factor would have eigenvalue of 0.9)
  2. An elbow can be observed after the 5th value when plotting number of components vs eigenvalues
  3. With 5 factors we are explaining more than 50% of variance

Using more factors would introduce more complexity in the analysis while explaining marginal variance. We could use fewer factors to simplify the analysis, but we would be missing factors with eigenvalue > 1, and would contain less than 50% of variance.

2. How should she associate each of the original 29 variables to the factors selected?

She should create a rotated set of factors using a rotation technique such as varimax, then drop all variables for a factor that have a coefficient less than a certain threshold (we have used 0.5).

For example, factor 5 is 0.71xQ1_21 + 0.70xQ1_28 + 0.59xQ1_29

Our Approach

Factor Analysis in 6 steps

  1. Confirm the data in metric

  2. Decide whether to scale or standardize the data

  3. Check the correlation matrix to see if Factor Analysis makes sense

  4. Develop a scree plot and decide on the number of factors to be derived

  5. Interpret the factors (consider factor rotations - technical but useful)

  6. Save factor scores for subsequent analyses
