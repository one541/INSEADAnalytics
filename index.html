Course Description

This course is about running data analytics projects and interpreting the results. You will also learn practical tools to implement the analytics projects discussed.

We will discuss applications of data analytics in a wide range of business cases, from Marketing, to Finance, to Operations among others. We will cover techniques such as factor analysis and positioning methods, discriminant analysis and targeting methods, and cluster analysis and segmentation methods. These techniques are useful for example in market segmentation and targeting or mapping market structure and product design.

Although it will not be a technical class, the material covered in the INSEAD core course Uncertainty, Data & Judgment or an introductory course on Statistics for Business is a prerequisite. The course has a heavy "hands-on" flavour: we will analyse datasets using standard open source (or other) data analytics software tools - which you will be able to directly use in your jobs. As we will be using these tools during class, participants are expected to come to class with a laptop.

What you will take away from this course:

Upon completing the course, you will, among others:

  • Become more aware of what business problems can be addressed with data analytics;
  • Be able to better identify new business applications of data analytics;
  • Better understand the promises and limitations of big data;
  • Understand the processes for analysing data and managing analytics projects;
  • Learn key analytic techniques that will help you to make better business decisions;
  • Gain some experience with the latest data analytics tools that are widely used in companies ranging from Google and Amazon to high tech start-ups and hedge funds.

The emphasis of the course will not be on the technical/mathematical aspects of data analytics, but rather on its practical use in organizations.

Class Group Project and the Sample Data Analytics Projects Page

For the class group project, every group (sizes of 3-5 people per group) is required to develop a data analytics case for which, if possible, data is also gathered and, when possible, shared. Sample guidelines for the project deliverable can be found here, while sample case study with data can be founds in the Data Analytics Projects page.

Course Sessions

For every session there are also technical readings to familiarize participants with some of the "classic" analytics methodologies and tools. These are only strating examples, there are many other methods and technologies we will talk about in class. Course sessions will mainly focus on the business uses and implications of such tools, in addition to clarifying the technical aspects of data analytics.

All participants are required to setup a github account and provide their username before the first session.

If you experience any issues using the class technologies, please post them at the "Issues" of the course main github page, after exploring any related past issues there.
